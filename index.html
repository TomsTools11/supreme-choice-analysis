7 Methodology

This analysis follows the GOAL Platform four-layer diagnostic framework. Layer 1 parsed the client's stated problem — zero leads on an active day — and identified it as an urgent volume issue requiring investigation. Layer 2 built a comparative trend table across four time periods (last month, this month, last week, this week). The diagnostic matrix (stable/up searches + declining win rate + declining impression share) confirmed Competitive Positioning as the primary root cause. Layer 3 drilled into source-level data, filtering to active sources with opportunities, calculating effective bids using the $2.26 ad group base bid, and classifying each source's problem type. Layer 4 calibrated all recommendations against the trailing 7-day RightPricing report for Home — CA. Target multipliers were calculated using the formula: Target Multiplier = (Target RPC / Base Bid) × 100, with a 5–10% buffer applied per the rounding rule to account for bid volatility. Position targets were set at 2nd or 3rd position based on volume impact potential and cost efficiency, with 1st position recommended only where the position spread is narrow (Tier 2).
